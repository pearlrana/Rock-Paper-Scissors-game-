# 🎮 Rock Paper Scissors

A simple Rock Paper Scissors game made with HTML, CSS, and JavaScript.

## 🕹️ How to Play
Choose Rock, Paper, or Scissors and play against the computer.  
The rules are simple:

- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  

## 🛠️ Built With
- HTML  
- CSS  
- JavaScript  

## ✨ About
Just a fun beginner project to practice JavaScript and DOM manipulation.

Enjoy the game! 🎉
